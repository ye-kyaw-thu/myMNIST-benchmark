# Data Information

We used the BHDD (Burmese Handwritten Digit Dataset) for our benchmarking experiments.  
To ensure fair comparison with previous work, we used the original training and testing splits provided by the dataset authors.  

The BHDD dataset can be accessed here:  
[https://github.com/baseresearch/BHDD](https://github.com/baseresearch/BHDD)
